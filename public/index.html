Welcome to the SanGuoSha English Reference!

This website is designed to be used as a reference for English speakers to while playing the Chinese version of SanGuoSha.

This website is designed to be viewed using a tablet computer such as an iPad, although it should work on any modern browser.

All tranlations and photos are from sanguoshaenglish.blogspot.com. Refer to this website for a walkthrough of the rules of SanGuoSha as well as more in-depth analysis of the cards and their relation to the Chinese epic, "Romance of the Three Kingdoms".
